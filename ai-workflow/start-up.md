# 🚀 Start-up Guide (Onboarding)

> **ยินดีต้อนรับสู่โปรเจกต์ js_tesseract_opencv**
> ไฟล์นี้คือ "จุดเริ่มต้น" สำหรับทุกคนที่เพิ่งเข้ามาในโปรเจกต์นี้ หรือเริ่ม Session ใหม่หลังห่างหายไปนาน
> กรุณาอ่านและกดไปยังลิงก์ตามลำดับด้านล่าง

---

## 📖 ลำดับการอ่านเอกสาร (Reading Order)

### 1. 🎯 เข้าใจเป้าหมายของโปรเจกต์ (What are we building?)

👉 **อ่านไฟล์:** `ai-workflow/prd.md`

- **ทำไมต้องอ่าน:** เพื่อรู้ว่าโปรเจกต์นี้ทำอะไร มีฟีเจอร์อะไรบ้าง ใครคือผู้ใช้งาน
- **สิ่งที่จะได้รู้:** Tech Stack, Core Features, ข้อจำกัดต่างๆ

### 2. 📍 เข้าใจสถานะปัจจุบัน (Where are we now?)

👉 **อ่านไฟล์:** `ai-workflow/hand-over.md`

- **ทำไมต้องอ่าน:** นี่คือไฟล์ส่งต่องาน ⚡ _(สำคัญมาก)_
- **สิ่งที่จะได้รู้:** Phase ปัจจุบัน, งานที่เสร็จ, งานค้าง, ปัญหา (Blocker)

### 3. ✅ ดูความคืบหน้า (Progress & Planning)

👉 **อ่านไฟล์:** `ai-workflow/checklist.md`

- **ทำไมต้องอ่าน:** ดูว่าโปรเจกต์ถึง Phase ไหนแล้ว
- **อยากดูลงลึก?** ดูที่โฟลเดอร์ `implement-plan-xx/`

### 4. 📐 เข้าใจโครงสร้าง (How is it built?)

👉 **อ่านไฟล์:** `ai-workflow/structure-file.md`

- **ทำไมต้องอ่าน:** เพื่อรู้โครงสร้าง Folder ทั้งหมด
- **เจาะลึก:** ดูไฟล์ `design-xx.md` ของแต่ละ stream

### 5. ✍️ กฎการทำงาน (Rules of the house)

👉 **อ่านไฟล์:** `ai-workflow/principle.md`

- **ทำไมต้องอ่าน:** เพื่อรักษาความสม่ำเสมอของมาตรฐานเอกสาร

---

## 🚦 สรุปสั้นๆ ถ้ารีบ

1. โปรเจกต์นี้ทำอะไร? -> **`prd.md`**
2. ต้องทำอะไรต่อ? -> **`hand-over.md`**
3. ถึง Phase ไหนแล้ว? -> **`checklist.md`**
